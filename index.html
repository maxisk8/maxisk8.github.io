BIENVENIDOS A TAGASSIST

Quienes somos?

Somos 4 alumnos de la Escuela E.P.E.T N1 llamados Bonini lucas, Piris da Mota Juan, Olivera Maximo y Szeliga Leonardo. Decidimos hacer este proyecto ya que nos cansamos de las largas filas y el sistema tardio de asistencia con el codigo de barra de la escuela, esta suma de circustancias nos inspiro a crear un nuevo sistema de asistencia NFC con el fin de dejar en el olvido los problemas del actual sistema de asistencia.

Como trabajamos?

Nos especializamos en el trabajo en equipo pero tambien trabajamos individualmente en el proyecto nos dividimos las tareas equitativamente a la habilidad personal de cada uno y en ciertas tareas donde se complica hacerlo de forma individual lo realizamos como equipo de trabajo gracias a eso podemos agilizar las tareas que realizamos, cada tarea tiene una fecha para ser finalizada y en caso de saber que no llegariamos a terminarla tendremos que pedir mas tiempo para poder terminarla

IDEA

Este proyecto surgió gracias a el sistema de asistencia de la escuela, Este sistema se basa en una lectura de código de barras para cargar la asistencia a una base de datos, el problema era la mala administracion del sistema haciendo que se genere largas filas y los alumnos marcarán asistencia tarde aunque llegaron a tiempo

Funcionalidad

Utilizamos un arduino 1 que nos proporcionara una plataforma estable y versatil para desarrollar este sismtea NFC.

Para la lectura de etiquetas NFC hemos seleccionado el lector RFID NFC PN 532 de 13.56Mhz este lector ofrece una alta compatibilidad con una amplia gama de tarjetas NFC estandar y proporciona un a interfaz sencilla para la comunicacion con el arduino
